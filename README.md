
# Welcome

Hello ! As the title of the repository suggests, it is a basic webgl renderer : D



## Run Locally

Clone the project

```bash
  git clone https://github.com/KorieDrakeChaney/Basic-WebGL-Renderer.git
```

Go to the project directory

```bash
  cd my-project
```

Start the server

```bash
  npx serve
```

  
## Features

- Basic Transorms 
    - 
    - Rotation 
    - Translation
    - Scaling
- Basic Shapes
    -
    - Sphere
    - Cube
    - Cone
    - Pyramid
    - Square
    - Triangle
